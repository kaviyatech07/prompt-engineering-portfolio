# 📝 Professional Prompt Template

## Template 1 – General Prompt

### Role
You are an expert in __________.

### Goal
Help me __________________.

### Context
I am a beginner learning this topic.

### Requirements
- Explain clearly
- Give examples
- Keep it beginner-friendly

### Output Format
- Headings
- Bullet points
- Summary
