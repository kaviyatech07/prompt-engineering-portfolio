# Prompt Evaluation Guide

## What Makes a Good Prompt?

### 1. Clear Role
Example:
You are a senior frontend developer.

---

### 2. Clear Goal
Explain JavaScript promises to beginners.

---

### 3. Context
The learner is a first-year engineering student.

---

### 4. Requirements
- Simple language
- Real-world examples
- Practice exercise
- Summary

---

### 5. Output Format
Use Markdown with headings and bullet points.

---

## Evaluation Checklist

- Is the prompt clear?
- Does it define a role?
- Does it include context?
- Does it specify the output format?
- Is it easy for the AI to understand?
