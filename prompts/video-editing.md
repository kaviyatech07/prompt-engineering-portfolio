# 🎬 Video Editing Prompts

## Prompt 1 – YouTube Intro

### Prompt
You are a professional video editor.

Create a 10-second YouTube intro for an AI technology channel.

Requirements:
- Modern animation
- Blue and black color theme
- Smooth transitions
- Cinematic music suggestion

### Purpose
Practice creating engaging intros.

---

## Prompt 2 – Instagram Reel

### Prompt
Create a 30-second Instagram Reel script explaining Artificial Intelligence.

Requirements:
- Hook in first 3 seconds
- Simple language
- Fast-paced editing suggestions
- Subtitle ideas

### Purpose
Learn short-form content creation.

---

## Prompt 3 – Tech Tutorial

### Prompt
Generate a storyboard for a 5-minute tutorial on GitHub for beginners.

Requirements:
- Scene-by-scene breakdown
- Camera angles
- Text overlays
- Background music ideas

### Purpose
Improve educational video planning.
