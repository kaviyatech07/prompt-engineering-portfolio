# Prompt Engineering Techniques

## 1. Zero-Shot Prompting

The AI is asked to perform a task without any examples.

### Example

Prompt:

```
Explain Machine Learning in simple words.
```

---

## 2. One-Shot Prompting

The AI receives one example before solving the task.

### Example

Input:

```
Apple → Fruit

Car → ?
```

Output:

```
Vehicle
```

---

## 3. Few-Shot Prompting

The AI is given multiple examples.

### Example

```
Dog → Animal
Rose → Flower
Car → Vehicle

Laptop → ?
```

Output

```
Electronic Device
```

---

## 4. Role Prompting

Assign a role to the AI.

### Example

```
Act as an experienced Python developer and explain decorators.
```

---

## 5. Chain of Thought Prompting

Encourage the AI to reason step by step.

### Example

```
Solve the problem step by step.
```

---

## 6. Self-Consistency Prompting

Generate multiple reasoning paths and choose the most consistent answer.

---

## 7. ReAct Prompting

Combine reasoning with actions to solve complex tasks.

---

## 8. Tree of Thoughts

Explore multiple solution paths before selecting the best one.

---

# Summary

| Technique | Best Use |
|-----------|----------|
| Zero-Shot | Simple tasks |
| One-Shot | One example |
| Few-Shot | Pattern learning |
| Role Prompting | Expert responses |
| Chain of Thought | Logical reasoning |
| Self-Consistency | Better accuracy |
| ReAct | Tool usage and reasoning |
| Tree of Thoughts | Complex problem solving |

---

## Conclusion

These techniques improve the quality, accuracy, and reliability of AI-generated responses and are widely used in modern prompt engineering.
