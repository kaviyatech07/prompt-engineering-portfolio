# Prompt Evaluation Guide

## Objective

This guide demonstrates how to evaluate and improve prompts for better AI responses.

---

# Example 1

## Initial Prompt

```
Write about Python.
```

### Problems

- Too broad
- No audience specified
- No expected format

---

## Improved Prompt

```
Explain Python programming for beginners in less than 150 words with three real-world applications.
```

### Result

- Clear
- Focused
- Beginner-friendly
- Better structured response

---

# Example 2

## Initial Prompt

```
Create a resume.
```

### Problems

- No role specified
- No experience level
- No formatting instructions

---

## Improved Prompt

```
Create a one-page resume for a B.Tech AI & Data Science student with Python, SQL, Machine Learning, Excel, and GitHub projects.
```

---

# Evaluation Checklist

| Criteria | Yes/No |
|-----------|---------|
| Clear objective | ✅ |
| Specific audience | ✅ |
| Output format mentioned | ✅ |
| Context provided | ✅ |
| Constraints included | ✅ |

---

# Best Practices

- Be specific.
- Define the audience.
- Mention the desired output format.
- Include constraints when needed.
- Review and refine prompts for better results.

---

## Conclusion

High-quality prompts produce more accurate, useful, and consistent AI-generated responses.
