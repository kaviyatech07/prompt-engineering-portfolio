# 📚 AI Study Planner

## Project Description

This mini project helps students create a personalized study schedule using AI prompts.

## Prompt

You are an experienced academic mentor.

Create a weekly study plan for a third-year AI & Data Science engineering student.

Requirements:
- 2 hours daily
- Python practice
- Machine Learning
- Web Development
- Revision on weekends

## Expected Output

- Daily timetable
- Weekly goals
- Monthly progress plan

## Skills Used

- Prompt Engineering
- Planning
- AI Assistance
