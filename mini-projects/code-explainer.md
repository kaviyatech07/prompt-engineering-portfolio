# AI Code Explainer

## Project Overview

This prompt helps beginners understand source code by explaining it in simple language.

---

## Prompt

```
Act as a Python instructor.

Explain the following Python code line by line.

For each line:
- Describe its purpose.
- Explain the syntax.
- Mention common beginner mistakes.

End with a short summary.
```

---

## Expected Output

- Line-by-line explanation
- Beginner-friendly language
- Learning tips
- Summary
