# 📄 AI Resume Generator

## Project Description

This project uses prompt engineering to generate an ATS-friendly resume for students and job seekers.

## Problem

Many students don't know how to create a professional resume.

## Prompt

You are an expert resume writer.

Create an ATS-friendly resume for a third-year AI & Data Science student.

Requirements:
- Professional Summary
- Skills
- Projects
- Education
- Certifications
- Internship
- Clean formatting

## Expected Output

- One-page resume
- ATS-friendly structure
- Professional wording

## Skills Used

- Prompt Engineering
- Resume Writing
- AI Assistance

## Future Improvements

- Generate resumes for different job roles
- Customize resumes based on job descriptions
